Beyond Zero to Mastery

The Complete Web Developer in 2020: ZTM

Course Content

  • Introduction
    In this section, you will be introduced with the course content and to the community.
  • How The Internet Works
    You'll learn how the internet works, inspect the Google, and what is the work of a developer.
  • History Of The Web
    You'll start from the history and get introduced with HTML, CSS and Javascript.
  • HTML 5
    You'll be learning the basic concepts of HTML such as tags, self closing tags, anchor tags.
  • Advanced HTML 5
    You'll be learning advanced concepts such as submitting forms.
  • CSS
    You'll get to know about CSS and some of the basics in CSS.
  • Advanced CSS
    You'll get to know some advanced concepts such as animations.
  • Bootstrap 4, Templates, And Building Your Startup Landing Page
    You'll learn to use bootstrap in your projects to make it look awesome.
  • CSS Grid + CSS Layout
    You'll learn one of the latest features of CSS known as CSS grid.
  • Career Of A Web Developer
    You'll know about the recent techs and how to cover your lack of experience.
  • Javascript
    You'll get to know the power of Javascript and implement your new power in a project.
  • DOM Manipulation
    You will learn what DOM is and how to manipulate it in order to create some cool websites.
  • Advanced Javascript
    You'll learn some advanced concepts in javascripts such as currying, composing etc.
  • ECMA Script Concepts
    Latest Javascript trends with concepts ranging from ES5, ES6 to ES9 & ES10.
  • Command Line
    You'll learn to use terminal/command prompt for the web development.
  • Developer Environment
    You'll configure your editor to make it look like a pro.
  • Git + Github + Open Source Projects
    You'll learn to contribute to an open source project by learning Git.
  • A Day In The Life Of A Developer
    A typical day of a developer.
  • NPM + NPM Scripts
    You'll get to know about NPM and how to setup NPM.
  • React.js + Redux
    You'll learn one of the upcoming technology known as React and Redux.
  • HTTP/JSON/AJAX + Asynchronous Javascript
    You'll get hands on in one of the important features of ES8.
  • Backend Introduction
    You'll be introduced with the backend.
  • APIs
    You'll be introduced with APIs and an optional exercise to explore more of it.
  • FINAL PROJECT: SmartBrain Front-End
    This is an ultimate part where you'll build the frontend for your final project.
  • Node.js + Express.js
    You'll get to know about Node and Express.
  • FINAL PROJECT: SmartBrain Back-End -- Server
    You'll build the backend for your final project.
  • Databases
    You'll learn about the databases.
  • FINAL PROJECT: SmartBrain Back-End -- Database
    You'll confiure your final project with the database you have learned in this course.
  • Production + Deployment
    Finally, deployment.
  • Where To Go From Here?
    Personalized Advice from the Instructor!
  • Bonus:Extra Bits
    Ask Me Anything videos by the Instructor!
  • Extra: For Windows Users
    Learn about installation guide for Windows users.
  • Extras
